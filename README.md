# 业务监控交互式数据看板 (Sales Dashboard)

基于 HTML5, CSS3 和 Vanilla JavaScript 构建的高性能交互式销售看板。支持 Excel 数据导入、多维下钻分析及环比增长自动计算。

## 核心特性

- **零后端依赖**：单 HTML 文件（dashboard_final.html）通过 CDN 加载 ECharts 和 SheetJS，无需服务器即可运行。
- **智能计算**：
  - **MoM (环比)**：支持“当月比上月”和“滚动 3 个月均值”两种口径，自动处理跨年（2025/2026）数据。
  - **YoY (同比)**：智能匹配跨年份指标。
- **动态交互**：
  - 树形结构下钻：支持从“省区 -> 办事处 -> 医院”等多级深度分析。
  - 多维过滤：支持月份、产品、省区等组合查询。
  - 排序与转换：支持金额与“最小包装数”一键切换，支持各指标动态排序。
- **全端适配**：深度优化 iOS (Safari/Edge) 和 Android 兼容性，支持手机端直接操作。

## 使用说明

1. 直接在浏览器中打开 `dashboard_final.html`。
2. 点击顶部的“导入数据源 (.xlsx)”按钮，选择符合模板格式的 Excel 文件。
3. 通过上方的过滤器和切换按钮进行数据分析。
